# Hướng dẫn tạo Template .docx cho Quote Generator

## Bước 1: Tạo file Word (.docx)

Mở Microsoft Word và tạo một document mới với nội dung như sau:

```
KẾ HOẠCH TRIỂN KHAI DỰ ÁN

Dự án: {{ten_web}}

Khách hàng: {{khach_hang}}

Thực hiện: Misty Team (Đại diện: Bảo Nhật - Zalo: 0914960029)

Ngôn ngữ & Công nghệ sử dụng: {{cong_nghe}}

Tổng chi phí ước tính: {{chi_phi}}

Thời gian triển khai: {{thoi_gian}}

1. MỤC TIÊU DỰ ÁN

Xây dựng website {{ten_web}}, cho phép:

• {{cho_phep_1}}

• Yêu cầu khác: {{yeu_cau_1}}

2. PHÂN QUYỀN NGƯỜI DÙNG

• User: {{user_1}}

• Admin: {{admin_1}}

3. THỜI GIAN THỰC HIỆN

• Giai đoạn 1: {{khoang_time_1}}
• Giai đoạn 2: {{khoang_time_2}}  
• Giai đoạn 3: {{khoang_time_3}}
• Giai đoạn 4: {{khoang_time_4}}

4. BẢO HÀNH

{{bao_hanh}}

---
Generated by Quote Generator
```

## Bước 2: Lưu file

1. Lưu file với tên: `clean_quote_template.docx`
2. Đảm bảo format là .docx (không phải .doc)
3. Không có formatting phức tạp (tables, images, etc.)

## Bước 3: Upload vào hệ thống

1. Vào Quote Generator
2. Click Edit template "Báo giá 1"  
3. Upload file `clean_quote_template.docx`
4. Hệ thống sẽ validate và detect placeholders

## Lưu ý quan trọng:

- Placeholders phải có format: `{{field_name}}`
- Không có spaces trong placeholder: `{{ten_web}}` ✅, `{{ ten_web }}` ❌
- Tên field phải match với database: `ten_web`, `khach_hang`, etc.
- File phải sạch, không có lỗi formatting
